# 📊 Análisis de Eficiencia en la Resolución de Casos

En entornos de atención al cliente multicanal, la eficiencia operativa impacta directamente en la experiencia del cliente y en los costos del servicio.  
Este proyecto analiza el desempeño del proceso de atención de casos con el objetivo de identificar oportunidades de mejora basadas en datos.

## 🧠 Objetivo

Evaluar el desempeño operativo para identificar:

- Factores asociados a cancelaciones  
- Tipos de casos con mayor tendencia a escalar  
- Equipos con mayor efectividad en resolución  
- Canales con mayor tiempo de atención  

El objetivo es detectar oportunidades de mejora en eficiencia operativa y experiencia del cliente.

## 🏢 Problema de Negocio

En entornos de atención multicanal, las organizaciones enfrentan:

- Altos tiempos de resolución  
- Escalamiento frecuente de casos  
- Cancelaciones de solicitudes  

Comprender qué factores influyen en estos resultados permite optimizar la gestión del servicio y mejorar la operación.

---

## ⭐ Hallazgos Clave ⭐

El análisis permitió identificar patrones relevantes en la operación:

- CDMX concentra el mayor número de cancelaciones.
- Los casos de prioridad media presentan mayor volumen de cancelación.
- Los casos relacionados con facturación y cancelación muestran mayor tendencia a escalar.
- El equipo especializado lidera en la resolución de casos.
- El canal Teléfono presenta el mayor tiempo promedio de resolución.
- Chat se posiciona como el canal más ágil de atención.
- Las etapas iniciales del flujo concentran mayor tiempo de permanencia.
